img2pdfについて
  フォルダに画像を複数格納し、その後このプログラムを実行後、ポップアップの手順に沿って進んでいき完了後
  フォルダに存在する画像全てをpdfの形で結合します。
----------------------------------------------------------------------------------------------------------------
originalCode(フォルダ)
  コード(Python)が記述されているファイル
----------------------------------------------------------------------------------------------------------------
Mac(フォルダ)
  distファイルにmacOS用の実行ファイルが格納されています。
----------------------------------------------------------------------------------------------------------------
img2pdfForWindows(フォルダ)
  distファイルにWindows用の実行ファイルが格納されています。
